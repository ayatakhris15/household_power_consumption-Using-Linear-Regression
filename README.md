Electricity Consumption Prediction using Linear Regression

This project predicts electricity consumption using Linear Regression based on historical numerical energy data. The goal is to demonstrate a complete machine learning pipeline including preprocessing, training, and evaluation.

Dataset

Source: Kaggle (UCI Machine Learning Repository)

Link: https://www.kaggle.com/datasets/uciml/electric-power-consumption-data-set

Type: Numerical time-series electricity data

Features Used

Voltage

Global Intensity

Sub Metering 1

Sub Metering 2

Sub Metering 3

Target: Electricity Consumption (continuous)

Methodology

Load and clean dataset

Handle missing values

Apply feature scaling

Split data (70% train, 30% test)

Train Linear Regression model

Predict consumption values

Evaluate using accuracy and confusion matrix

Results

Accuracy: 99.44%

Model shows strong performance with balanced predictions.
